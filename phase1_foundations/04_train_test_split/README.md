# Day 4: Train/Test Split and Overfitting
This experiment explores the concept of train/test split and overfitting in machine learning. It's essential to understand when to use these techniques to evaluate and improve model performance.

* Train/Test Split: dividing data into training and testing sets
* Overfitting: when a model performs well on training data but poorly on new data
* Model Evaluation: assessing model performance on unseen data

How to run: `python main.py`
