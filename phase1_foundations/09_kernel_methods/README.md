# Kernel Methods
Kernel methods is an approach used in machine learning to solve non-linear problems using the kernel trick, which allows us to operate in higher dimensional spaces without explicitly computing the coordinates of the data points. This technique is particularly useful when dealing with complex, non-linear relationships between variables.

* Kernel trick
* Non-linear problem solving
* Higher dimensional spaces
* Efficient computation
