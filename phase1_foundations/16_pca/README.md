# Principal Component Analysis (PCA) Experiment
This is the 16th day of the ML experiment series, focusing on Principal Component Analysis (PCA), a technique used to reduce dimensionality in datasets while retaining most of the information. It is useful when dealing with high-dimensional data and wanting to visualize or analyze it in a lower-dimensional space.

* Dimensionality reduction
* Feature extraction
* Noise reduction
* Data visualization

To run the experiment, use `python main.py`
